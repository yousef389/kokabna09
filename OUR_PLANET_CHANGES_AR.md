# سجل تجهيز نسخة كوكبنا

- توحيد اسم المشروع إلى `kokabna_private_app`.
- ضبط التطبيق على العربية واتجاه RTL كامل.
- تعريب النصوص الظاهرة وإزالة الجملة الإنجليزية من الواجهة.
- إضافة ملفات Android ناقصة: `gradle.properties`، وملفات manifest الخاصة بالـ debug/profile.
- تحسين إعدادات Gradle للعثور على Flutter SDK من `local.properties` أو `FLUTTER_ROOT`.
- إضافة `analysis_options.yaml` خفيف.
- تحسين معاينة الملفات: صور، فيديو، صوت، وفتح خارجي.
- تنظيف أسماء الملفات ومسارات التخزين قبل الرفع.
- تعريب بعض رسائل Cloud Functions والإشعارات.
- تحديث README وتعليمات بناء APK.

## ملاحظة البناء داخل هذه البيئة

بيئة العمل هنا لا تحتوي على Flutter SDK أو Dart، لذلك لا يمكن إنتاج APK فعلي داخل الجلسة. المشروع جاهز للبناء على جهاز يحتوي Flutter وAndroid SDK.
